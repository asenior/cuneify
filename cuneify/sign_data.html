Cuneiform sign search

A tool to help find a cuneiform sign by appearance.
Use the controls to restrict the signs shown. Stroke counts are computed with the NeoAssyrian 'CuneiformNA' font and will differ for the same sign shown in other fonts. The 'name' is the character name contained in the font file. Readings come from the ORACC sign list (aligned with the unicode code point).

Strokes includes both long strokes and winkelhaken.
